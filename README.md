# Python-OOP-Interview-Questions
Python OOP Interview Questions


Basic Concepts
1. What is Object-Oriented Programming (OOP)?

OOP היא גישת תכנות שבה בונים את הקוד סביב אובייקטים.
כל אובייקט מייצג ישות שיש לה:

נתונים (attributes / fields  ) לדוגמא CAR יכול להכיל COLOR, BRAND, SPEED
התנהגויות (methods) drive()  / stop()

What is the difference between a class and an object?

Class = תבנית / blueprint / שבלונה

Object = מופע אמיתי שנוצר מהמחלקה


What is a constructor in Python?

ה־constructor הוא המנגנון שמופעל כשיוצרים אובייקט חדש מהמחלקה.
בפייתון בדרך כלל מתכוונים ל־__init__.

What does the method init do?

המתודה __init__ מאתחלת את האובייקט אחרי שהוא נוצר.

היא בדרך כלל:

מקבלת ערכים התחלתיים

שומרת אותם בתוך האובייקט

מבצעת אתחול של שדות

What is the purpose of the self keyword?

self מייצג את האובייקט הנוכחי.

בעזרת self אפשר לגשת ל:

שדות של האובייקט

מתודות של האובייקט
